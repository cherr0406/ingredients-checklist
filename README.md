# (WIP)Ingredients Checklist

url: <https://ingredients-checklist.vercel.app/>

## Description

Web サイトからレシピを取得し、買い物リストを作成するアプリケーションです。

To run the development server, clone the repo and run:

```bash
npm run dev
```
